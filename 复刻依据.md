# Path App `+` 动效复刻依据

## 复刻目标

本次只复刻 `Path 2.0` 中左下角点击 `+` 后展开的 action menu 交互动效，不扩展到整套产品 UI。

## 可验证依据

1. TechCrunch 在 2011-11-29 对 `Path Two` 的描述明确写到：
   - “点击左下角的 `+` 按钮，会弹出 6 个选项图标。”
   - 来源：<https://techcrunch.com/2011/11/29/paths-second-iteration-is-less-photosharing-and-more-everything-sharing/>

2. The Next Web 在 2011-12-04 对这个交互的拆解给了更细的运动描述：
   - 按钮位于左下角。
   - `+` 点击后会旋转成 `x`。
   - 6 个圆形子按钮沿扇形“飞出”。
   - 子按钮会“逆时针旋转四分之一圈”并带“轻微回弹”。
   - 收回时速度更快、更脆。
   - 来源：<https://thenextweb.com/news/look-out-tab-bar-get-ready-for-paths-sharing-ui-to-be-everywhere>

3. Proto.io 的教程页直接把这个交互称为 `Path app menu animation`，证明该交互后来被大量按同一特征复刻：
   - 来源：<https://support.proto.io/hc/en-us/articles/220679508-Prototyping-Path-Part-1>

4. Sparanoid 公开了一个 `Path 2.0 Flyout Menu using CSS` 的复刻版，并在页面与仓库中明确说明这是 `Path 2.0` 飞出菜单实现：
   - Demo：<https://sparanoid.com/lab/path-menu/>
   - Repo 页面：<https://github.com/sparanoid/path-menu>

5. `PathMenu` 的 iOS 复刻项目给出了接近真实菜单的参数范围：
   - `animationDuration = 0.5`
   - `farRadius = 110`
   - `nearRadius = 90`
   - `endRadius = 100`
   - `menuWholeAngle = π - π/5`
   - `rotateAngle = -π/2 + π/10`
   - 来源：<https://cocoapods.org/pods/PathMenu>

## 本地实现与依据对应关系

1. 左下角主按钮
   - 依据：TNW 对原交互的文字描述。
   - 实现：[`style.css`](/Users/wutao/美柚/点滴/path效果还原/style.css) 中 `.flyout-wrap` 与 `.flyout-btn`。

2. `+` 旋转成 `x`
   - 依据：TNW 描述；Sparanoid demo 里主按钮 `span` 使用旋转切换。
   - 实现：[`style.css`](/Users/wutao/美柚/点滴/path效果还原/style.css) 中 `.flyout-btn.btn-rotate span`。

3. 子按钮数量
   - 原始依据：TechCrunch 明确写出 “six option icons”。
   - 当前页面：为了满足当前需求，已从原始 `6` 个改成 `4` 个，保留原交互的扇形展开、级联延迟与回弹逻辑。
   - 实现：[`index.html`](/Users/wutao/美柚/点滴/path效果还原/index.html) 中 `ul.flyout > li` 共 4 项。

4. 扇形展开与等分角度
   - 原始依据：Sparanoid demo 的 0/18/36/54/72/90 度分布；与 TNW 的“quarter turn counter-clockwise”描述一致。
   - 当前页面：因按钮数改为 4 个，改成 `0/30/60/90` 度，保持同样的四分之一圆扇形观感。
   - 实现：[`style.css`](/Users/wutao/美柚/点滴/path效果还原/style.css) 中 `.flyout > li:nth-of-type(...)`。

5. 展开时上冲后回弹
   - 依据：TNW 的 “slight bounce”；Sparanoid demo 的 `expand` 关键帧。
   - 实现：[`style.css`](/Users/wutao/美柚/点滴/path效果还原/style.css) 中 `@keyframes expand`。

6. 收起更快、更脆
   - 依据：TNW 的 “traveling faster, giving it a snappier feel”；Sparanoid demo 的 `contract` 时长短于展开。
   - 实现：[`style.css`](/Users/wutao/美柚/点滴/path效果还原/style.css) 中 `animation: contract 0.35s` 与 `@keyframes contract`。

7. 子按钮自旋
   - 依据：TNW 的“逆时针四分之一圈”；Sparanoid demo 的 `spinX-expand / spinX-contract`。
   - 实现：[`style.css`](/Users/wutao/美柚/点滴/path效果还原/style.css) 中 `spin1` 到 `spin4` 系列关键帧。

8. 级联延迟
   - 依据：Sparanoid demo 的 0.04s 级差。
   - 当前页面：因按钮数量减少，延迟链路同步压缩，最后一项延迟为 `0.12s`。
   - 实现：[`style.css`](/Users/wutao/美柚/点滴/path效果还原/style.css) 中每个 `nth-of-type` 的 `animation-delay`。

## 说明

1. 这次本地页面使用的是可直接验证的公开复刻实现参数和 2011 年原始报道中的运动描述。
2. Path 原始私有工程代码不可公开获取，所以“100%”这里采用的是：
   - 以当年的原始媒体描述为行为依据；
   - 以公开的 Path 同款复刻源码为参数依据；
   - 让本地交互在动作轨迹、节奏、层级和反馈上与公开可验证版本一致。
